About the project

The climate is not visualized.
The climate commands.

ATMOSPHERICA is a Machine Learning system that transforms real-time atmospheric data into generative abstract painting. Each artwork is unique. Every formal decision comes from real data.

The idea

There is a fundamental difference between visualizing data and translating it into visual language. A chart shows temperature. ATMOSPHERICA turns it into color, into form, into composition.

The system calculates which climate variable has the highest normalized value at any given moment — temperature, wind, humidity, pressure, clouds or PM2.5 — and that variable deforms the entire visual system. It does not only control its own form: it reshapes the global composition, orients the canvas, compresses or expands all other elements.

The result is a painting that feels different depending on the climate. A day of extreme heat in July produces something radically different from a windy autumn day or a cloudy morning in January — without reading a single number.

Visual grammar

Each climate variable has its own family of forms, its own fixed color and its own behavior depending on the data value.

●
Temperature
Concentric circles. Universal fixed color: blue=cold · green=temperate · orange=heat · red=extreme. Dominant: they invade the canvas without hierarchy.
∿
Wind
Bézier curves oriented according to the real geographic wind direction. Dominant: lines crossing the entire canvas. The canvas rotates slightly.
◯
Humidity
Triangles at low humidity. Diffuse ovals with halos at high humidity. Dominant: blurred edges, dense fog sensation.
▬
Pressure
Rectangles. High pressure: horizontal parallel bands, architectural composition. Low pressure: tilted fragments at the edges.
◆
Clouds
Flattened diamonds. Dominant: visual weight at the top, reduced contrast, the sky collapses over everything.
·
PM2.5
Dots and violet haze. Dominant: contaminates all previous layers. Pollution makes the rest of the image unreadable.

Technical pipeline

01

Real-time data ingestion

OpenWeatherMap API — temperature, pressure, wind, humidity, clouds. Air Pollution API — PM2.5, NO₂, O₃. Data updated on every execution.

02

Normalization and visual mapping

Each variable is normalized using historical ranges from Seville. The mapper computes 15 visual parameters: color, density, fragmentation, opacity, stroke direction.

03

Predictive ML model

Random Forest trained with 5 years of ERA5 data (Copernicus). Predicts the probability of extreme events the next day. AUC 0.854 · F1 0.694. The prediction visually alters the piece.

04

Pictorial generation

HTML5 Canvas with JavaScript. Deterministic seed — the same climate always produces the same image. The dominant variable deforms the entire visual system.
